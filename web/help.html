تواجه تقطيعًا أو بطئًا؟

غالبًا يكون السبب تعطّل التسريع الرسومي في المتصفّح. اتبع خطوات متصفّحك التالية لتفعيله — وستلاحظ تحسّنًا فوريًا في سلاسة البث المباشر.

Google Chrome
تفعيل التسريع الرسومي
  1. افتح الإعدادات (Settings).
  2. انتقل إلى:
    System → Use graphics acceleration when available
  3. فعّل الخيار.
  4. أعد تشغيل المتصفّح.
للتحقّق افتح chrome://gpu ويجب أن يظهر:
Video Decode: Hardware accelerated
🦁
Brave
تسريع + إيقاف الدرع
  1. افتح:
    brave://settings/system
  2. فعّل Use graphics acceleration when available.
  3. أعد تشغيل المتصفّح.
🦁 إذا استمرّت المشكلة (مهم لـ Brave):
  1. اضغط أيقونة 🦁 Brave Shield بجانب شريط العنوان.
  2. أوقف Shields لهذه الصفحة.
  3. حدّث الصفحة.
للتحقّق افتح brave://gpu
Microsoft Edge
تفعيل التسريع الرسومي
  1. افتح:
    edge://settings/system
  2. فعّل Use hardware acceleration when available.
  3. أعد تشغيل المتصفّح.
Mozilla Firefox
تفعيل التسريع الرسومي
  1. افتح:
    about:preferences
  2. ابحث عن قسم Performance.
  3. أزل علامة Use recommended performance settings.
  4. فعّل Use hardware acceleration when available.
  5. أعد تشغيل المتصفّح.
ملاحظة مهمة

البث يستخدم تقنية WebRTC منخفضة التأخير، لذلك يُنصح باستخدام أحدث إصدار من المتصفّحات التالية، مع إبقاء التسريع الرسومي مفعّلًا للحصول على أفضل أداء وجودة بث.
